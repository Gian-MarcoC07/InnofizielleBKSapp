# Inofizielle BKS App
Eine simple App für meine Schule, gemacht mit React-Native und Expo.

## Features
- Shortcuts (Webmail, Cloud, Menüplan, Stundenplan, Schulnetz und BKS Website)
- Notenrechner
- Todo-Liste
